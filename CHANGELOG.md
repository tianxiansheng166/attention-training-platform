### 首次发布
专注力星球v1.0版本发布！

### 新增功能
- 🎮 **时空密码** - N-Back工作记忆训练游戏
- 🎮 **星际拦截** - Stop-Signal抑制控制训练游戏
- 🎮 **密林寻踪** - 视觉搜索注意力训练游戏
- 🎮 **要塞守望** - 警觉性持续注意力训练游戏
- 🧠 **专注力初始定级** - 自动评估专注力水平
- 🛠️ **智能任务工坊** - AI任务分解功能
- 📊 **成长报告** - 综合训练数据报告

### 技术特性
- 前端：HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- 后端：Node.js + Express.js
- AI集成：DeepSeek LLM API
- 部署：Nginx反向代理 + PM2进程管理
